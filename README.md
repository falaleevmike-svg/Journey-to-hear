# Путешествие к сердцу 💕

Романтический 2D пиксельный платформер для браузера.

## Описание

Игра о паре, которая путешествует по миру, преодолевая препятствия и собирая воспоминания. 
Продолжительность: 1.5-2.5 часа.

## Управление

- **Стрелки / WASD** — движение
- **Пробел / W** — прыжок (двойной прыжок доступен)
- **E** — взаимодействие
- **Shift** — спринт
- **Esc** — пауза / меню

## Уровни

1. **ДВФУ** (Владивосток) — стартовый уровень, кампус университета
2. **Москва** — городской уровень, метро, крыши
3. **Китай** (Чунцин, Чэнду, Чжанцзяцзе) — три подуровня
4. **Турция** (Стамбул, Олюдениз) — базар, парапланы
5. **Офис** — комедийный уровень, лабиринт кабинок
6. **Побережье** — финальный уровень, закат

## Технологии

- HTML5 Canvas
- JavaScript (ES6+)
- Чистый JS без фреймворков
- localStorage для сохранения прогресса

## Запуск

Откройте `index.html` в браузере или запустите локальный сервер:

```bash
python -m http.server 8000
```

## Особенности

- 🎮 Управление парой как единым целом
- 💕 Милые анимации взаимодействий (руки, подсаживание, подтягивание)
- 📷 Система воспоминаний (24 штуки)
- 🏖️ Parallax-эффекты фонов
- 🎵 Уникальная музыка для каждой локации
- 💾 Автосохранение в localStorage
- 📱 Поддержка тач-управления для мобильных
